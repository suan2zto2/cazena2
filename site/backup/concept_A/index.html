    안 A — 카제나 방식 × 패키지 RPG

    기본 정보

    항목 내용
    장르 턴제 카드 RPG
    플랫폼 Steam (패키지)
    예상 플레이타임 1회차 50~60시간
    파티 구성 3인 고정
    전투 방식 STS 기반 카드 전투 + 3인 파티 덱 / 카오스 콘텐츠에서 덱 완성 후 영구 유지
    유사 게임 Slay the Spire, Across the Obelisk, Monster Train

    핵심 컨셉

    카제나 방식의 핵심인 "카오스에서 덱을 완성해 유지한다" 를 패키지 RPG 구조에 이식한다.

    • 덱은 런마다 리셋되지 않고 게임 전체에 걸쳐 영구 성장
    • 카오스는 덱을 빌딩하는 전용 콘텐츠 구간
    • 완성된 덱으로 스토리 챕터 보스와 메인 진행을 돌파
    • 3인 파티 각각이 독립 덱을 보유 → 파티 구성이 덱 전략의 출발점

    코어 루프

    스토리 챕터 진입
    ↓
일반 전투 구간 (현재 덱으로 진행)
    ↓
카오스 구역 개방 ──────────────────────────────┐
    ↓                                           │
카오스 내 전투 클리어 → 카드 획득 / 선택        │
    ↓                                           │
덱 점진적 완성 ──────────────────────────────── ┘
    ↓
챕터 보스 도전 (완성된 덱 사용)
    ↓
보스 클리어 → 신규 챕터 개방 + 신규 카드 풀 해금
    • 카오스는 챕터마다 1~2개 배치, 진행하지 않아도 챕터 클리어는 가능
    • 단, 카오스를 통해 덱을 강화할수록 보스 난이도가 크게 낮아짐
    • 카오스를 모두 완료하면 해당 챕터 전용 강화 카드 1장 추가 획득

    전투 시스템

    기본 구조

    • 3인 파티, 캐릭터마다 개인 덱(12~15장) 보유
    • 전투 시작 시 3개 덱이 합산된 공용 드로우 풀 생성
    • 매 턴 5장 드로우, 공용 에너지 3 으로 원하는 카드 사용

    카드 사용과 에너지

    • 카드마다 에너지 비용 1~3 상이
    • 같은 캐릭터 카드를 연속 사용하면 집중 보너스 발동 (다음 카드 비용 -1)
    • 에너지를 다 쓰거나 턴 종료를 선택하면 적 행동

    적 행동 예고

    • 적은 매 턴 시작 시 다음 행동을 예고 (공격 수치, 버프, 광역 여부)
    • 예고를 읽고 방어 카드 배치 / 우선 처치 대상 결정이 핵심 전술

    파티 시너지 카드

    • 특정 조합의 캐릭터가 파티에 함께 있을 때만 드로우 풀에 추가되는 시너지 카드 존재
    • 시너지 카드는 개인 덱에 넣을 수 없고, 파티 편성에 따라 자동 삽입

    성장 시스템

    덱 성장 (메인 성장 축)

    • 카오스 / 전투 보상으로 카드 획득 → 덱에 추가 or 보관함 저장
    • 카드 업그레이드: 재료 소비 → 수치 강화 또는 효과 추가
    • 보관함은 챕터 사이 거점에서 자유롭게 편집

    캐릭터 레벨

    • 전투 경험으로 레벨 상승 → 패시브 특성 1개 선택 (3가지 중 택 1)
    • 패시브는 덱 방향성을 강화하는 효과 위주
    • 예: "에너지 1 초과 사용 시 드로우 +1", "집중 보너스 발동 시 공격력 +20%"

    유물 시스템

    • 카오스 보스 처치 / 특수 이벤트에서 유물 획득 (STS의 렐릭과 유사)
    • 유물은 전투 규칙 자체를 바꾸는 강력한 패시브
    • 3인 파티 전체에 영향을 주는 유물 vs. 특정 캐릭터 전용 유물 구분

    패키지 게임 특성 반영

    요소 설계 방향
    엔딩 구조 챕터 클리어 누적 → 최종 챕터 보스 격파 시 엔딩. 수집 요소 달성도에 따라 엔딩 분기
    덱의 서사 연결 카드는 단순 스킬이 아닌 캐릭터의 과거 기억/능력. 강화할수록 캐릭터 서사 공개
    재플레이 유인 2회차 신규 난이도 + 1회차에서 선택하지 않은 카드 루트 탐색
    난이도 조절 카오스 선택 여부로 자연스럽게 난이도 자가 조절
