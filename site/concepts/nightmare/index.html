    안 3 — 나이트메어 클럽

    악몽을 상품으로 파는 지하 클럽에서 각자의 트라우마를 짊어진 여성들이 꿈과 현실의 경계를 탐험하는 심리 호러 RPG

    기본 정보

    항목 내용
    가제 림버스 카제나 : 나이트메어 클럽
    장르 턴제 RPG
    플랫폼 Steam (패키지)
    예상 플레이타임 1회차 50~60시간
    참고작 카오스 제로 나이트메어

    세계관

    사람들은 돈을 내고 타인의 악몽을 체험하는 지하 클럽을 찾는다. 쾌락, 스릴, 혹은 자신의 상처를 다른 이의 꿈 속에서 발산하기 위해.

    그 클럽의 이름이 카제나(KAZENA).

    하지만 최근 고객들이 악몽에서 깨어나지 못하는 사건이 연이어 발생한다. 플레이어는 카제나 소속 드림 클리너 — 악몽 속으로 잠입해 갇힌 의식을 구출하고, 내부의 괴물을 처치하는 해결사 팀의 리더다.

    "악몽은 거짓말을 하지 않습니다. 그게 문제예요."

    핵심 특장점

    AI 리소스 활용

    • 악몽 세계의 비현실적 배경과 왜곡된 캐릭터 비주얼 → AI 생성 리소스 활용 폭 가장 넓음
    • 각 캐릭터의 트라우마 공개 씬 이 성우 연기의 핵심 감정 구간
    • 공포 + 미소녀 조합 → 틈새 시장 공략

    악몽 구조

    각 챕터는 특정 고객의 악몽 세계를 탐험하는 방식으로 전개된다.

    악몽 구성
├── 현실 파트    : 사건 접수 + 팀 브리핑
├── 잠입 파트    : 악몽 세계 탐색 (규칙이 챕터마다 다름)
├── 심층 파트    : 트라우마 핵(核) 접근 + 보스 전투
└── 각성 파트    : 결말 + 클리너 자신의 악몽 일부 공개

    캐릭터 구조

    역할 설명 전투 스타일
    클리너 리더 팀을 이끄는 베테랑, 자신의 악몽을 봉인 중 균형형 / 지휘 버프
    몽상가 악몽 세계의 규칙을 직관적으로 파악 환경 이용 / 트릭
    투사 공포를 분노로 덮는 타입, 과격한 성격 근접 특화 / 광역
    분석가 트라우마 구조를 논리로 해석하는 냉정한 인물 약점 탐지 / 디버프
    미확인 DLC 추가 예정 -

    전투 시스템

    • 악몽 규칙 가변 시스템 : 챕터마다 전투 환경 규칙이 달라짐 (예: 특정 속성 봉인, 회피 불가 등)
    • 공포 게이지 : 아군 공포 수치가 임계점을 넘으면 패닉 상태 → 행동 불능 리스크
    • 트라우마 약점 : 보스의 핵심 트라우마를 분석해 약점 공략 → 스토리와 전투가 연동
    • 동료 악몽 챕터 : 메인 스토리 후반, 팀원 자신의 악몽에 진입하는 특수 챕터 등장

    수익 모델

    항목 내용
    기본 패키지 본편 (메인 악몽 챕터 + 팀원 악몽 챕터)
    회차 플레이 요소 수집형 악몽 기록, 히든 엔딩, 고난이도 악몽 재도전
    DLC 신규 클리너 캐릭터 + 해당 캐릭터의 전용 악몽 챕터

    강점 / 약점

    강점 약점
    세 안 중 가장 독창적 → 시장 차별화 공포 요소 → 호불호 갈림
    AI 비주얼 리소스 활용 범위 최대 악몽 규칙 가변 → 밸런스 설계 난이도 높음
    성우 연기 임팩트 씬 가장 많음 레퍼런스 인지도 카오스제로 < 림버스
    DLC 캐릭터별 독립 악몽 → 확장 구조 명확
