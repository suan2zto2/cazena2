    게임 개요

    🎬 게임 소개 슬라이드 보기

    항목 내용
    게임명 테세라위버 (Tessera Weaver)
    장르 턴제 카드 RPG
    플랫폼 Steam (패키지)
    예상 플레이타임 1회차 50~60시간
    파티 구성 최대 3인 출전 (보유 캐릭터 3인 이상 가능)
    전투 방식 제한된 슬라이드 횟수 안에 핸드 카드 숫자 타일을 보드에 준비 → 액션 버튼으로 일괄 실행 → 핸드 교체 → 적 행동
    유사 게임 Slay the Spire (카드), Puzzle Quest (퍼즐+RPG), Into the Breach (행동 예고+대응)

    핵심 컨셉

    제한된 슬라이드 안에서 핸드를 얼마나 완성할 것인가.

    • 매 턴 시작 시 핸드에 카드가 드로우되며, 카드마다 발동에 필요한 숫자 타일이 지정됨
    • 플레이어는 제한된 슬라이드 횟수 안에 보드를 조작해 핸드 카드에 맞는 타일을 준비
    • 액션 버튼을 누르면 보드 위의 타일이 행동 재화로 변환되고, 해당 재화로 발동 가능한 핸드 카드가 실행됨. 핸드 전체를 버린 뒤 새로 드로우
    • 액션 버튼을 누를 때마다 적 행동 카운트가 1 감소하며, 카운트 0 도달 시 예고 행동 실행
    • 높은 숫자일수록 강력한 카드 — 그러나 높은 숫자를 만들려면 슬라이드가 더 필요

    핵심 딜레마:

    "핸드에 64짜리 필살 카드가 있다. 만들려면 슬라이드가 5번은 더 필요한데 남은 횟수가 3번밖에 없다. 64를 포기하고 4·8·16만 확실히 실행할까, 아니면 도박을 걸까?"

    코어 루프

    스토리 챕터 진행
    ↓
전투 돌입 — 보드 초기화 + 핸드 드로우
    ↓
보드 슬라이드 (턴당 횟수 제한, 적 카운트에 영향 없음)
    ↓
핸드 카드 숫자에 맞는 타일 준비 → 액션 버튼
    ↓
타일 있는 카드 실행 / 없는 카드는 버려짐 → 핸드 전체 교체
    ↓
모든 적 카운트 -1 → 카운트 0인 적 행동 실행
    ↓
[반복] 적 전멸 → 전투 승리 → 카드 보상 획득
    ↓
거점 귀환 → 덱 편집 / 캐릭터 강화
    ↓
다음 챕터 진입

    패키지 게임 특성

    요소 설계 방향
    엔딩 구조 챕터 클리어 누적 → 최종 챕터 보스 격파 시 엔딩. 수집 요소에 따라 엔딩 분기
    학습 곡선 초반은 낮은 카운트 적 + 낮은 숫자 카드 위주 → 후반 챕터에서 짧은 카운트 적 + 높은 숫자 카드 요구
    전략 깊이 슬라이드 횟수 계산 + 핸드덱 우선순위 + 적 행동 예고 대응이 동시에 맞물림
    재플레이 유인 덱 방향 (저숫자 다발 vs. 고숫자 필살) + 패시브 선택에 따라 완전히 다른 플레이 스타일

    콘텐츠 구성

    메인 시나리오

    총 8챕터. 각 챕터는 약 30개 스테이지의 연속 플레이로 구성되며, 슬레이 더 스파이어와 유사하게 경로를 선택하며 진행한다.

    요소 내용
    챕터 수 8챕터 (순차 해금)
    스테이지 수 챕터당 약 30개 (경로 선택 포함)
    보스 챕터당 3개. 보스 클리어마다 시나리오 씬 진행
    경로 선택 중간 스테이지는 플레이어가 경로를 선택해 진행
    패배 처리 챕터 클리어 실패. 진행한 스테이지 수에 비례해 경험치 획득
    클리어 보상 다음 챕터 해금 + 경험치 획득

    챕터 흐름

    챕터 시작
    ↓
스테이지 선택 (경로 분기)
    ↓
전투 반복 → 중간 보스 (시나리오 씬)
    ↓
전투 반복 → 중간 보스 (시나리오 씬)
    ↓
전투 반복 → 최종 보스 (시나리오 씬)
    ↓
챕터 클리어 → 다음 챕터 해금

    서브 시나리오

    메인 시나리오 진행 중 특정 조건을 충족하면 해금된다.

    유형 내용
    캐릭터 시나리오 각 동료 캐릭터의 개인 스토리
    이벤트 시나리오 단편 이벤트성 에피소드
    해금 조건 시나리오마다 개별 조건 부여 (챕터 클리어, 특정 캐릭터 동반 등)
    확장 계획 추후 DLC로 서브 시나리오 추가

    서브 시나리오 전체 클리어가 추가적인 장기 목표로 기능한다.

    랭킹 모드

    점수 경쟁 모드. 메인 시나리오와 별개로 제공되며, 제한된 조건에서 최고 점수를 경쟁한다.
