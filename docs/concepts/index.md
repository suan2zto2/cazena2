## 안별 비교

| 항목 | [안 A — 카제나 방식 × 패키지 RPG](concept_A.md) | [안 B — 스테이지 클리어 × 덱 리셋 로그라이크](concept_B.md) | [안 C — 2048 에너지 보드 × 덱 카드 RPG](../backup/concept_C.md) |
|------|------|------|------|
| **핵심 키워드** | 카오스에서 덱 완성 → 영구 유지 | 스테이지마다 덱 리셋 + 메타 진행 | 2048 보드로 에너지 생성 → 스킬 발동 |
| **덱 리셋 여부** | 없음 (게임 전체 유지) | 스테이지 클리어마다 리셋 | 없음 (게임 전체 유지) |
| **전투 흐름** | 드로우 → 에너지 소모 → 카드 사용 | 드로우 → 에너지 소모 → 카드 사용 | 보드 슬라이드로 수치 조합 → 일치하는 스킬 발동 |
| **전략 핵심** | 카오스 활용 여부 + 덱 방향성 설계 | 런마다 덱을 빠르게 완성하는 판단력 | 보드 상태 읽기 + 손패 계획 + 연계 타이밍 |
| **파티 상호작용** | 공용 드로우 풀 + 파티 시너지 카드 | 공용 드로우 풀 + 캐릭터 사망 시 카드 소실 | 공용 손패 + 연속 발동 시 연계 스킬 |
| **성장 방식** | 덱 강화 / 캐릭터 패시브 / 유물 | 런 내 성장 + 메타 해금 (영구 누적) | 덱 강화 / 보드 확장 패시브 / 유물 |
| **엔딩 구조** | 챕터 클리어 누적 → 최종 보스 | 메타 진행 + 스토리 조각 수집 → 트루 엔딩 | 챕터 클리어 누적 → 최종 보스 |
| **유사 게임** | Slay the Spire, Across the Obelisk | Slay the Spire, Hades, Darkest Dungeon | Slay the Spire, Puzzle Quest, 2048 |
